  • Request
  • Response

    UUID representing account

Body

Media type: application/json

Type: object

Properties
  • parkingArea: required(string)

    Code representing parking area

  • receptionMethod: required(one of postal, email)

    Valid reception method

  • postalCode: (string)

    Canadian postal code to send sticker to, required if reception method is postal

  • email: (string)

  "parkingPeriod": "monthly"
}

Sent via email:

{
  "parkingArea": "Zone1",
  "receptionMethod": "email",
  "email": "spamd.ul@ulaval.ca"
}
