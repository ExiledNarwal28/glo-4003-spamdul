  • Request
  • Response

URI Parameters

  • accountId: required(string)

HTTP status code 200

Requested user's bills

Body

Media type: application/json

Type: array of object

Items: Bill

  • billId: required(string)

    UUID representing bill

  • billType: required(one of offense, parkingSticker, accessPass)

    Valid bill type

    Description of bill

  • amountDue: required(number)

    Total amount due for bill

  • amountPaid: required(number)

Example:

[

    "amountDue": 100.0,
    "amountPaid": 0.0
  },
  {
    "billId": "e78e9362-93a8-46a7-badf-d849b0d09c4c",
    "billType": "parkingSticker",
    "description": "Purchase of a parking sticker with code a2cc479c-6889-45cc-8f6c-97eb64127399",
    "amountDue": 80.0,
    "amountPaid": 20.0
  }
]

  • Request
  • Response

URI Parameters

  • accountId: required(string)

    UUID representing account

Body

Media type: application/json

Type: object

Properties
  • licensePlate: required(string)

    Valid license plate of car

  • manufacturer: required(string)

    Manufacturer of car

  • model: required(string)

    Model of car

  • year: required(number)

    Year of car

  • consumptionType: required(one of greedy, economical, hybridEconomonical, superEconomical, zeroPollution)

  "model": "Camry",
  "year": 2006,
  "consumptionType": "economical"
}

/api/users/{accountId}/accessPasses post

Collection of access passes for user

post

Add a access pass to user

post /api/users/{accountId}/accessPasses

Add a access pass to user

  • Request
  • Response

URI Parameters

  • accountId: required(string)

    UUID representing account

Body

Media type: application/json

Type: object

Properties
  • licensePlate: (string)

    License plate for car. No license plate simply means no car is associated to the access pass.

  • accessDay: required(one of monday, tuesday, wednesday, thrusday, friday, saturday, sunday)

    Day of access pass validity

Examples:

With car:

{
  "licensePlate": "123 ABC",
  "accessDay": "monday"
}

Without car:

{
  "accessDay": "monday"
}

HTTP status code 201

Code for newly added access pass

Body

Media type: application/json

Type: object

Properties
  • accessPassCode: required(string)

    Code representing access pass

Example:

{
  "accessPassCode": "PASS-12345"
}

/api/users/{accountId}/parkingStickers post

Collection of parking stickers for user

post

Add a parking sticker to user

post /api/users/{accountId}/parkingStickers

Add a parking sticker to user

  • Request
  • Response

URI Parameters

  • accountId: required(string)

    UUID representing account

Body

Media type: application/json

Type: object

Properties
  • parkingArea: required(string)

    Code representing parking area

  • receptionMethod: required(one of postal, email)

    Valid reception method

  • postalCode: (string)

    Canadian postal code to send sticker to, required if reception method is postal

  • email: (string)

    Email to send sticker to, required if reception method is email

Examples:

Send via postal carrier:

{
  "parkingArea": "Zone1",
  "receptionMethod": "postal",
  "postalCode": "A1E 2E1"
}

Sent via email:

{
  "parkingArea": "Zone1",
  "receptionMethod": "email",
  "email": "spamd.ul@ulaval.ca"
}

HTTP status code 201

Code for newly added parking sticker

Body

Media type: application/json

Type: object

Properties
  • parkingStickerCode: required(string)

    Code representing parking area

Example:

{
  "parkingStickerCode": "PARK-123"
}

HTTP status code 400

Error adding parking sticker to user

HTTP status code 404

User not found
